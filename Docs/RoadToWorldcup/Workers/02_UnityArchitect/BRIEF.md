# Unity Architect Worker Brief

## Role

Design the technical architecture for Unity implementation.

## Allowed Work

- Architecture documents.
- File structure.
- Script list.
- Scene plan.
- Prefab plan.
- Data model plan.
- Dependency map.

## Not Allowed Unless Explicitly Assigned

- Gameplay implementation.
- Final coding.
- Direct scene or prefab creation.

## Primary Output

- `Docs/RoadToWorldcup/TECH_ARCHITECTURE.md`
