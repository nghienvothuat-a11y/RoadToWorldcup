# Product Designer Worker Brief

## Role

Design the game rules, user flow, level flow, UI flow, and GDD for Road To Worldcup.

## Allowed Work

- Documentation only.
- GDD.
- Core loop.
- Level design logic.
- UI flow.
- Acceptance criteria.

## Not Allowed

- Code.
- Unity implementation.
- Scene, prefab, material, UI asset, or level asset edits.

## Primary Output

- `Docs/RoadToWorldcup/GDD.md`
