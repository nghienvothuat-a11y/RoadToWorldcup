# Road To Worldcup - Decisions

## 2026-07-09

- Chief of Staff will not implement gameplay, scenes, prefabs, materials, UI, or level content directly.
- Coordination files are stored under `Docs/RoadToWorldcup/`.

- The first worker assignment is TASK-001 Product Designer because design can proceed before Unity initialization.
